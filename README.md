# 规划开发中...

## 简介

`vue-pure-visual` 是一款开源免费且开箱即用的可视化大屏模版。完全采用 `ECMAScript` 模块（`ESM`）规范来编写和组织代码，使用了最新的 `Vue3`、
`Vite`、`Bun`、`ECharts`、`TypeScript`、`Tailwindcss` 等主流技术开发
